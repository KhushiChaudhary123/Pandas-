# Pandas Data Analysis Project

This project demonstrates data analysis using Python and Pandas.

## 📊 Features
- Data cleaning and preprocessing
- Handling missing values
- Exploratory Data Analysis (EDA)
- Extracting meaningful insights from datasets

## 📁 Files Included
- Pandas.ipynb → Main analysis notebook
- Student.csv → Dataset used
- my_analysis.xlsx → Processed data

## 🚀 Learning Outcome
Through this project, I learned how to work with real-world data using Pandas and transform raw data into useful insights.
